# :cut_of_meat: Carnes Abdul
Sitio web de la Carniceria Abdul, ubicado en Toltén, Araucania, Chile
## :hammer: Stack
- Next.js 15
- Tailwind CSS v4
- React Icons
- Google Maps
## :toolbox: Instalación
npm install
npm run dev
## :robot: Deploy
Desplegado en [Vercel](https://carnes-abdul.vercel.app)
## :pushpin: Secciones
- Hero con video de fondo
- Catálogo de productos
- Historia del negocio
- Conctacto con mapa, horarios de atención y redes sociales
